# gaokao-zhiyuan · 高考志愿填报顾问 skill

把"一个分数"变成"一张能用的冲稳保志愿表"。一个 agent skill + 打分引擎：**冲稳保梯度 + 五维匹配度 + 就业方向(绿/红牌) + 决策纪律**，输出文本或一张可转发给家长的「录取参考单」HTML。

> **核心纪律：先查官方数据再推荐，查不到的院校位次绝不编。** 编错了考生会滑档——这是一个不骗人的工具。

## 它是什么 / 不是什么
- ✅ 是：一套**方法论 + 打分引擎**。SKILL.md 指挥 agent 现查教育部权威数据，引擎做冲稳保和五维打分。
- ❌ 不是：一个塞满全国录取数据的离线库。仓库里 `data/sample.csv` 是**北京物化生演示数据**（位次/就业/薪资均为演示值），**不可直接用于真实填报**。正式用时按 `SKILL.md` 工作流现查官方历年投档位次。

## 特性
- **冲稳保**：冲 = 院校位次比考生高 1~12%；稳 = ±5%；保 = 低 5~25%。
- **五维匹配度成绩单**：分数匹配 / 就业前景 / 兴趣适配 / 城市地域 / 院校平台（0-100，缺输入显示"待填"，不瞎给分）。
- **就业方向标签**：🟢绿牌急需 / 🚀国家急需可冲 / ✅工科稳就业 / 🛡师范保底 / 🟡就业偏弱(生化环材) / 🔴红牌预警。
- **硬门槛**：选科符合度 + 排斥方向（`--reject` 一票否决，师范类不被学科名误伤）。
- **兴趣用排除法**：`--accept`/`--reject` 表达能接受/排斥，集外降权不剔除；只有家境宽裕才放开成加分（不把普通家庭孩子推向冷门）。
- **决策纪律**：就业倒推看中位数、城市优先、不可替代性/500强/10年后三镜。
- **HTML 报告**：自包含「录取参考单」模板，移动端友好、可转发家长。

## 用法
```bash
# 文本
python scripts/recommend.py --province 北京 --combo 物化生 --rank 28593 \
  --city 一线 --accept 工科,计算机 --reject 生物,化学 --grad-school no

# 生成 HTML 报告
python scripts/recommend.py --province 北京 --combo 物化生 --rank 28593 \
  --city 一线 --accept 工科 --format html --out report.html
```
完整工作流（含怎么现查官方数据）见 [`SKILL.md`](SKILL.md)。

## 数据来源
- 录取位次 / 控制线：教育部阳光高考 <https://gaokao.chsi.com.cn>、各省考试院。
- 就业绿/红牌：麦可思就业蓝皮书。急需方向：教育部红黄牌提示制度。出处见 [`references/employment-direction.md`](references/employment-direction.md)。

## 方法论
决策框架采用行业通行的**就业导向志愿原则**：就业倒推看中位数（看中间 50% 毕业生，不看顶尖个案）、城市优先、专业不可替代性、理工选专业·文科选学校。

## 免责声明
本工具仅供参考，**不构成官方录取承诺**。一切以考生所在省教育考试院官方一分一段表与历年投档数据为准。
