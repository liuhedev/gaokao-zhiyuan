# gaokao-zhiyuan

高考志愿填报顾问 —— 把一个分数变成一张能用的「冲稳保」志愿表。

一个 agent skill 加一个打分引擎：先查官方录取数据，再做冲稳保分档、五维匹配度评分和就业方向标注，最后输出文本或一张可转发给家长的「录取参考单」HTML。

核心纪律：先查官方数据再推荐，查不到的院校位次绝不编造。

## 关于数据

仓库内 `data/sample.csv` 是北京物化生的演示数据，位次、就业率、薪资均为示例值，不能直接用于真实填报。正式使用时，按 `SKILL.md` 的工作流现查当年当省的官方历年投档位次。

## 特性

- 冲稳保分档：冲（院校位次比考生高 1–12%）、稳（±5%）、保（低 5–25%）。
- 五维匹配度：分数匹配、就业前景、兴趣适配、城市地域、院校平台，各 0–100；缺输入则标「待填」，不瞎给分。
- 就业方向标注：绿牌急需 / 国家急需 / 工科稳就业 / 师范保底 / 就业偏弱 / 红牌预警。
- 选科门槛与兴趣排除：不符合选科要求的直接剔除；用 `--accept` / `--reject` 表达能接受、排斥的方向。
- 调节器：风险偏好、家庭经济、读研意愿，按需调整排序权重。
- HTML 报告：自包含「录取参考单」模板，移动端友好。

## 用法

```bash
# 文本
python scripts/recommend.py --province 北京 --combo 物化生 --rank 28593 \
  --city 一线 --accept 工科,计算机 --reject 生物,化学

# HTML 报告
python scripts/recommend.py --province 北京 --combo 物化生 --rank 28593 \
  --city 一线 --format html --out report.html
```

完整工作流（含如何现查官方数据）见 `SKILL.md`。

## 数据来源

录取位次与控制线取自教育部阳光高考（gaokao.chsi.com.cn）与各省考试院；就业方向标注参考公开就业蓝皮书与教育部就业提示。详见 `references/employment-direction.md`。
